Purpose of the Website

This website will model covalent molecules using the pubchem database. You can either type the molecular formula or press the buttons on the periodic table.


Using the 3D Molecular Viewer

Left Click and drag to rotate the 3D model. On trackpad, use two fingers and pinch or stretch in order to zoom in or out. On a mouse, use the scroll wheel to control zoom.


Why is the Molecular Formula So Weird?

The molecular formula that is shown uses a system called the Hills System, where the number of carbons is written first, then the number of hydrogens, and then the other elements in alphabetical order. If there are no carbons then all elements are written in alphabetical order.
